    * Installation de `xgboost <https://xgboost.readthedocs.io/en/latest/>`_ pour Windows :
      ``pip install pymyinstall`` et ``pymy_install xgboost``,
      see also
      `Build xgboost 0.4a30 for Python 3.4 and 3.5 <http://www.xavierdupre.fr/app/pymyinstall/helpsphinx//blog/2016/2016-04-04_build_xgboost_04a30.html>`_

    **Installation sous Windows**, essayer dans l'ordre :

    #. ``conda install <module>``
    #. Regarder sur la page `Unofficial Windows Binaries for Python Extension Packages <http://www.lfd.uci.edu/~gohlke/pythonlibs/>`_
       si elle contient le module souhaité. Si oui, il y a de grandes chances que
       le module nécessite un compilateur C. Essayer ``pip install <wheel>``.
    #. Enfin, essayer ``pip install <module>``.

    Si cela ne marche pas, installer
    `Visual Studio 2015 Community <https://www.visualstudio.com/fr-fr/products/visual-studio-community-vs.aspx>`_.
